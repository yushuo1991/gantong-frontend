# 感统训练中心 - 前端展示系统

## 📖 项目简介

这是一个**纯前端**的儿童感觉统合训练中心展示系统，专为演示和客户展示设计。系统提供完整的感统评估、训练指导和知识学习功能，所有数据存储在浏览器本地。

### ✨ 主要特性

- 🎯 **智能测评系统**：科学的感统能力评估，自动生成报告
- 🏋️ **训练指南**：针对不同感觉系统的训练方法
- 📚 **专业知识库**：深入浅出的感统理论知识
- 👨‍👩‍👧 **家长支持**：在线咨询、社区交流、培训课程
- 📱 **响应式设计**：完美支持手机、平板、电脑
- 🎨 **现代化UI**：美观的渐变色设计和流畅动画

### 🔧 技术特点

- ✅ 纯HTML + JavaScript实现，无需后端
- ✅ 使用localStorage本地存储数据
- ✅ 无数据库依赖，部署简单
- ✅ 现代化CSS3设计，AOS动画效果
- ✅ 完全独立，不影响其他项目

## 🚀 快速开始

### 本地预览

1. 直接用浏览器打开 `index.html`
2. 或使用简单HTTP服务器：
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

然后访问：http://localhost:8000

### 部署到服务器

详细部署步骤请查看：[快速部署指南.md](./快速部署指南.md)

**快速命令**（Linux/Mac）：
```bash
chmod +x deploy-gantong.sh
./deploy-gantong.sh
```

**Windows PowerShell**：
```powershell
.\Deploy-Gantong.ps1
```

## 📊 功能说明

### 1. 智能测评系统

- 支持3个年龄段（0-3岁、3-6岁、6岁以上）
- 6大感觉系统评估（触觉、前庭觉、本体觉、视觉、听觉、注意力）
- 自动计算分数并给出专业解读
- 生成可下载的HTML格式评估报告
- 数据保存在浏览器本地，保护隐私

### 2. 训练指南

- 详细的训练方法说明
- 分类清晰的训练项目
- 适合家庭实施的训练方案

### 3. 专业知识

- 感统理论基础知识
- 常见问题解答
- 专业术语解释

### 4. 家长支持

- 在线咨询入口
- 家长社区交流
- 培训课程信息

## ⚙️ 配置说明

### Nginx配置

项目已提供完整的Nginx配置文件：`gantong-nginx.conf`

**子域名配置**（推荐）：
- 域名：gantong.yushuo.click
- 根目录：/www/wwwroot/gantong

**子路径配置**：
- 路径：/gantong
- 根目录：/www/wwwroot/gantong

### DNS配置

如使用子域名，需在DNS服务商添加A记录：
```
类型: A
主机: gantong
记录值: [服务器IP地址]
TTL: 600
```

## 🔒 安全与隐私

- ✅ 所有评估数据**仅存储在用户浏览器本地**
- ✅ **不上传任何数据**到服务器
- ✅ **不收集用户信息**
- ✅ 部署在**独立目录**，与其他项目完全隔离
- ✅ 建议配置**HTTPS证书**提升安全性

## ⚠️ 重要说明

### 这是展示版本

1. **数据存储**：数据仅保存在浏览器本地（localStorage）
2. **数据持久性**：清除浏览器缓存会丢失数据
3. **多用户**：每个用户的数据独立，不共享
4. **生产环境**：如需完整功能，建议开发后端系统
